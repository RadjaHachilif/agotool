    Resources

    The resources used for this web-service are automatically updated every month.
    GO (basic) and GO-slims are retrieved from http://purl.obolibrary.org/obo/go/.
    UniProt-keywords are retrieved from http://www.uniprot.org/.
    Since the information quality of GO assocations differs between taxa (e.g. manually curated vs. Inferred from Electronic Annotation), a brief overview of the GO-association-resources used for this web-tool follows.
    UniProt provides
    A.) Species specific filtered resources:
    "species-specific annotation sets using the UniProtKB Complete Proteome and Reference Proteome sets that have undergone an additional electronic annotation filtering to remove redundancy." These are retrieved from ftp://ftp.ebi.ac.uk/pub/databases/GO/goa. This web-tool uses the gene_association.goa_"species" files. "(based on UniProtKB complete proteome) These files contains all GO annotations and information for a species subset of proteins in the UniProt KnowledgeBase (UniProtKB) and for entities other than proteins, e.g. macromolecular complexes (IntAct Complex Portal identifiers) or RNAs (RNAcentral identifiers). These files may provide annotations to more than one protein per gene. The protein accessions included in these files are all Swiss-Prot entries for that species plus any TrEMBL entries that have an Ensembl DR line. The TrEMBL entries are likely to overlap with the Swiss-Prot entries or their isoforms. If a particular entity is not annotated with GO, then it will not appear in this file (www.uniprot.org)." We are using "species specific filtered resources" for the following taxa:
    • Arabidopsis thaliana (Thale cress, TaxID: 3702)
    • Danio rerio (Zebrafish, TaxID: 7955)
    • Drosophila melanogaster (Fly, TaxID: 7227)
    • Gallus gallus (Chicken, TaxID: 9031)
    • Homo sapiens (Human, TaxID: 9606)
    • Mus musculus (Mouse, TaxID: 10090)
    • Rattus norvegicus (Rat, TaxID: 10116)
    • Saccharomyces cerevisiae (strain ATCC 204508 / S288c) (Yeast, TaxID: 559292 not 4932)
    • Sus scrofa (Pig, TaxID: 9823)
    B.) UniProt unfiltered resources:
    This is retrieved from: ftp://ftp.ebi.ac.uk/pub/databases/GO/goa/UNIPROT/gene_association.goa_uniprot.gz "This file contains all GO annotations and information for proteins in the UniProt KnowledgeBase (UniProtKB) and for entities other than proteins, e.g. macromolecular complexes (IntAct Complex Portal identifiers) or RNAs (RNAcentral identifiers). If a particular entity is not annotated with GO, then it will not appear in this file (www.uniprot.org)." To the best of our knowledge, there are no species specific filtered (gene association) resources are available for:
    • Chlamydomonas reinhardtii ('Chlamy', TaxID: 3055)
    • Equus caballus (horse, TaxID: 9796)
    • Medicago truncatula (barrel medic, TaxID: 3880)
    • Oryza sativa subsp. japonica (rice, TaxID: 39947)
    Therefore we are using "UniProt unfiltered resources" for these taxa.
